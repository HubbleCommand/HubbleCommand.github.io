This is a joke plugin that came to me in a dream, literally.
I, for some reason, was coding, and I kept sneezing.
Every time I sneezed, all my opened files would have random indentation.

It was infuriating.

So, for anyone that wants to live my dream, [here you go](https://github.com/HubbleCommand/sneeze).
At random intervals, the currently opened file will have each line's indentation changed.

If you happen to sneeze, there is also a button to force this effect.

I refuse to publish it to the vscode marketplace like I did my run plugin.
